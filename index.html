WFT · fase 2 — de 16 gekozen stijlen

Per stijl bovenaan de volledige video (klik = groot, mét Nederlands geluid). Daaronder het storyboard: start- en eindframe per scène, de clip (8s, Veo — Sanne/presentator spreekt Nederlands + natuurlijke beweging), en brug-clips (morph) tussen scènes waar dat past.

De twee scripts

A / B / C — "Van cursist naar vertrouwde adviseur" (3 scènes): S1 de vraag → S2 de opleiding (module-trap → certificering) → S3 het advies (handdruk). Elke clip = 8s Veo waarin Sanne Nederlands spreekt (lip-sync) met natuurlijke beweging. B/C krijgen brug-clips (morph) tussen scènes; A = match-cut.

P — presentator (4 segmenten, NL gesproken): set + presentator identiek. Elke clip = de presentator spreekt Nederlands (Veo native lip-sync) met wisselende camera/zoom en animerende paneel-content. Tekst: presentator.html.
