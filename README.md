# WFT stijlverkenning

Statische presentatie van de animatiestijlen (3 scènes × meerdere presets).
De media (video's + stills) worden geladen vanaf een publieke Supabase-bucket;
deze repo bevat alleen de HTML-pagina's.

- `index.html` — overzicht + vergelijking
- `realistisch.html` / `hybride.html` / `infomercial.html` / `presentator.html` — per categorie
